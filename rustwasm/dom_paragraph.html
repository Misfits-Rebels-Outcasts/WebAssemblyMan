HTML DOM paragraph Object for Rust WebAssembly

Create a paragraph Object

You can create an <paragraph> object using the document.create_element method.


    let paragraph = document.create_element("paragraph")
    .unwrap()
    .dyn_into::<web_sys::HtmlParagraphElement>()
    .unwrap();

{% include aa1.html %}
Access a paragraph Object

You can access an <paragraph> object using the document.get_element_by_id method.


    let button = document.get_element_by_id("my-paragraph")
    .unwrap()
    .dyn_into::<web_sys::HtmlParagraphElement>()
    .unwrap();
